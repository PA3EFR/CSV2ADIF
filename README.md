Het programma is een tot windows executable gemaakt programma van Github user Stiles96.

Ik heb daartoe het SLN programma via Visual Studio geconverteerd naar een windows executable.

Unzip het packet en dubbelklik op "CSV to ADIF Convertor.exe".

Nog enkele tips (ook in de Tips.txt):

2. Velden die je niet gebruikt: onderaan zetten in de lijst (kan zowel in ADIF als in CSV zijn)
